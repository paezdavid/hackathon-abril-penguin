Como usar la API

Python

Para utilizar los endpoint se debe utilizar la biblioteca Requests para con el metodo get para extraer los datos del endpoint que desea utilizar 

 data = requests.get("URL/penitenciarias/{id}").json()

cURL

Para utilizar un endpoint en una terminal, se debe utilizar 'curl' para realizar el m√©todo GET y extraer los datos del endpoint deseado y te tira dentro de la terminal el json. 

 curl -X GET "URL/penitenciarias/{id}" | jq

JavaScript

Para utilizar un endpoint en JavaScript (Node.js), se debe utilizar la biblioteca 'axios' para realizar el m√©todo GET y extraer los datos del endpoint deseado. 

 
const axios = require('axios');

axios.get(`URL/penitenciarias/${id}`)
  .then(response => {
    const data = response.data;
    console.log(data);
  })
  .catch(error => {
    console.error(error);
  });
